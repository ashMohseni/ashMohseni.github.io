Pizzeria


Pizzeria is a JavaFX application










This program asks the user to sign in to their account, and once they are in their account, they can build the pizza they prefer to. They'll also be able to view the receipt at the same time on the right side. If you would like to learn more about the sign-in / sign-up program and how it works, you can view its repository from here.




You can view the source code for this project from here.



How to setup this project










  • Step1: Download the JavaFx SDK from here!
  • Step2: Add the SDK to your libraries in your IDE
  • Step3: From project propperties go to `Libraries/Compile` and add the JavaFx library to the `Classpath`
  • Step4: From project propperties go to `Libraries/Run` and add the JavaFx library to the `Modulepath`
  • Step5: Add the following to your VM Options : `--add-modules javafx.controls,javafx.fxml,javafx.graphics`
  • Step6: Check off the Compile on Save option and you're good to go!!
